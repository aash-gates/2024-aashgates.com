About Me

I have been passionate about technology since the Windows Vista era, earning a reputation as a reliable tech enthusiast among my peers for my ability to solve technical challenges. I am deeply passionate about cloud computing, virtualization, networking, and contributing to the open-source community. With over 36,000 code contributions, my GitHub account ranks among the most active on the platform.

My journey also includes publishing a research paper on building an affordable server for source code control using Raspberry Pi. This project not only validated my research but also won the Best Project Award and received funding from the Karnataka State Council for Science and Technology.

Beyond coding, I am an avid tech enthusiast who stays updated with the latest trends and innovations. I enjoy participating in hackathons, contributing to open-source projects, exploring emerging technologies, playing video games, and spending quality time with family and friends.
